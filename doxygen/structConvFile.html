Detailed Description

File-backed convolution parameters.

Weight files are read sequentially in output-major order. For normal 2D convolution the layout is [O][I][K][K]; for 1D convolution it is [O][I][K]. Bias files contain one scalar per output channel. The same type is also used by depthwise convolution, where weights and biases are ordered by channel.
