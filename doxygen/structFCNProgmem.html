Detailed Description

AVR PROGMEM-backed fully-connected parameter bundle.

Use this when fully-connected weights and optional biases are stored in AVR flash memory and must be read with pgm_read_float_far(). weight_far is a far flash address, typically produced with pgm_get_far_address(weights), and points to row-major [O][I] float values. bias_far points to n_outputs float biases, or is 0 to use zero bias.
