2D Transpose Convolution

Memory-backed transpose convolution uses packed [I][W][W] input, packed [O][Vt][Vt] output, and weights in [O][I][K][K] order.

The output width formula is (W - 1) * S + K - 2 * P + OP.

Copy one square convolution kernel from PROGMEM into RAM.

Copies K * K consecutive floats starting at base from w into the row-major destination buffer kernel. On AVR, values are read through noodle_pgm_float(); on other targets the source pointer is read normally.

Parameters
wPROGMEM-backed packed weight array.
baseIndex of the first kernel value in w.
KKernel width and height.
kernelDestination buffer with room for K * K floats.

uint16_t W,
uint16_t K,
uint16_t S,
float * output )

Apply valid 1D pooling and write the result to memory.

K <= 1 is treated as identity pooling and copies input unchanged. If S is 0, the pooling stride defaults to K. This helper computes mean pooling when NOODLE_POOL_MODE == NOODLE_POOL_MEAN; otherwise it computes max pooling.

Parameters
inputInput vector with W floats.
WInput length.
KPooling window size.
SPooling stride, or 0 to use K.
outputDestination buffer with room for the pooled vector.
Returns
Output length, or W for identity/no pooling.
